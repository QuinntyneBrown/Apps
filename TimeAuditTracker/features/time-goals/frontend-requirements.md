# Frontend Requirements - Time Goals

## Components
- Goal Creation Wizard: Multi-step form for goal setup
- Goal Dashboard: Progress cards, milestone tracker, on-track indicators
- Milestone Celebration: Achievement modal with confetti animation
- Goal History: Completed and abandoned goals with lessons learned

## User Flows
1. Create Goal: Click "New Goal" → Enter description → Set target metrics → Set deadline → Save
2. Track Progress: View dashboard → See progress bars → Review on-track status
3. Celebrate Milestone: Achieve milestone → See celebration screen → Share achievement
4. Abandon Goal: Select goal → Click abandon → Enter reason → Confirm

## Visualizations
- Progress bars for each goal
- Timeline showing milestone markers
- Trend line for goal progress over time
- On-track vs off-track indicator badges

# Frontend Requirements - Comparison & Reporting

## Components
- Period Selector: Dual date range pickers for comparison
- Side-by-Side View: Parallel visualizations of two periods
- Ideal Allocation Editor: Sliders/inputs to define ideal time distribution
- Variance Display: Color-coded variance indicators (green/red)
- Report Export Dialog: Format selection, customization options

## User Flows
1. Compare Periods: Select period 1 → Select period 2 → Click "Compare" → View results
2. Set Ideal Allocation: Define ideal percentages per category → Compare with actual → See gaps
3. Export Report: Click export → Select format → Customize → Download

## Visualizations
- Side-by-side bar charts for period comparison
- Deviation chart showing improvements/regressions
- Alignment gauge for ideal vs actual
- Heat map for priority gaps

# Frontend Requirements - Schedule Optimization

## Components
- Suggestion Cards: Optimization recommendations with benefit estimates
- Before/After Visualizer: Side-by-side schedule comparison
- Time Block Creator: Recurring event setup wizard
- Calendar View: Weekly calendar with protected time blocks
- Impact Dashboard: Optimization effectiveness metrics

## User Flows
1. Review Suggestions: View recommendations → See expected benefit → Accept or reject
2. Create Time Block: Click "Add Block" → Define activity → Set recurrence → Set protection level → Save
3. Measure Impact: View implemented optimizations → See before/after metrics → Review ROI

## Visualizations
- Calendar heat map showing time blocks
- Before/after comparison charts
- ROI metrics dashboard
- Benefit estimation gauges

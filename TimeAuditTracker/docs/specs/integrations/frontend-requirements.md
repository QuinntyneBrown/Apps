# Frontend Requirements - Integrations

## Components
- Integration Setup Wizard: OAuth flow, permission requests
- Sync Status Dashboard: Connected services, last sync time
- Conflict Resolution Interface: Choose between calendar and logged time
- App Usage Breakdown: Charts showing time by application
- Integration Settings: Enable/disable, sync frequency

## User Flows
1. Connect Calendar: Click "Add Integration" → Select provider → Authorize → Sync
2. Resolve Conflict: See conflict warning → Review both entries → Choose or merge
3. Review App Usage: View imported data → Categorize apps → Approve import

## Visualizations
- Integration status indicators
- Conflict comparison view
- App usage pie chart
- Sync history timeline

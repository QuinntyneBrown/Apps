## UI/UX Notes

### Visual Design
- Use color-coded category badges
- Streak flames should be prominent and animated
- Cards should have subtle shadows for depth
- Use icon + text for better recognition
- Consistent spacing (8px grid system)

### Interactions
- Swipe left on habit card for quick archive (mobile)
- Long-press for quick menu (mobile)
- Drag to reorder habits
- Pull to refresh habit list
- Haptic feedback on completion

### States
- Loading: Skeleton screens
- Empty: Encouraging onboarding message
- Error: Clear error messages with retry
- Success: Toast notifications
- No results: Helpful suggestions

### Accessibility
- All interactive elements min 44x44px
- High contrast mode support
- Screen reader labels
- Keyboard shortcuts (desktop)
- Focus indicators

### Performance
- Virtual scrolling for 50+ habits
- Optimistic UI updates
- Cached data with smart refresh
- Lazy load archived habits

# Quality Assurance - Frontend Requirements

## Pages/Views

### 1. Inspections
**Route**: `/projects/{id}/inspections`
- Inspection checklist form
- Area-by-area rating system
- Defect photo capture
- Inspection history timeline
- Quality trend charts

### 2. Appraisals
**Route**: `/projects/{id}/appraisals`
- Appraisal record form
- Appraiser directory
- Value tracking over time
- Insurance documentation link
- Certificate storage

### 3. Show Results
**Route**: `/projects/{id}/shows`
- Show history list
- Award trophy case
- Score tracking charts
- Judge feedback display
- Show calendar integration

## Components
- InspectionChecklist with scoring
- QualityRatingStars
- DefectMarker for photo annotation
- AwardDisplay with badges
- TrendChart for quality improvement

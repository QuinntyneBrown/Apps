# Documentation - Frontend Requirements

## Pages/Views

### 1. Photo Gallery
**Route**: `/projects/{id}/gallery`
- Grid/timeline view toggle
- Filter by stage/date
- Before/after comparison viewer
- Bulk upload with drag-drop
- Photo slideshow
- Download/share options

### 2. Journal
**Route**: `/projects/{id}/journal`
- Journal entries list
- Rich text editor for new entries
- Mood tracking
- Attach photos to entries
- Export journal as PDF
- Search entries

### 3. Vehicle History
**Route**: `/projects/{id}/history`
- Timeline of previous owners
- Production details
- Original specifications
- Modification history
- Documentation upload
- Rarity information

## Components
- PhotoUploader with preview
- ImageEditor for cropping
- BeforeAfterSlider comparison
- JournalEditor with rich text
- TagInput for photo tagging
- TimelineView for history

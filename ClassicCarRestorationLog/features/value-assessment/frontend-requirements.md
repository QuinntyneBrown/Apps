# Value Assessment - Frontend Requirements

## Pages/Views

### 1. Valuation Tracker
**Route**: `/projects/{id}/value`
- Current value estimate
- Value history chart
- Comparable sales search
- ROI calculator
- Investment breakdown

### 2. Investment Analysis
**Route**: `/projects/{id}/investment`
- Total invested display
- ROI percentage
- Profit/loss calculation
- Time vs money invested
- Break-even analysis

### 3. Insurance Manager
**Route**: `/projects/{id}/insurance`
- Policy information form
- Coverage amount tracker
- Premium history
- Policy renewal reminders
- Document storage

## Components
- ValueChart showing appreciation
- ROIGauge with percentage
- ComparableSalesTable
- InvestmentBreakdown pie chart
- InsuranceReminderCard

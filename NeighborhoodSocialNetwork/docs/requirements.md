# Requirements - Neighborhood Social Network

## Overview
Connect neighbors and build community through local events, recommendations, and mutual support. The application enables neighborhood organizing, safety awareness, and local resource sharing.

## Core Features

### 1. Neighbor Directory
- FR-ND-001: Verify residence address before account activation
- FR-ND-002: Create neighbor profile with interests and skills
- FR-ND-003: Connect with nearby neighbors
- FR-ND-004: View neighborhood directory map

### 2. Community Events
- FR-CE-001: Create neighborhood events with RSVP
- FR-CE-002: Organize block parties with permit workflow
- FR-CE-003: Track event attendance
- FR-CE-004: Share event photos and highlights

### 3. Local Recommendations
- FR-LR-001: Review local businesses and services
- FR-LR-002: Recommend service providers
- FR-LR-003: Vouch for existing recommendations
- FR-LR-004: Request service recommendations

### 4. Safety & Alerts
- FR-SA-001: Report safety concerns
- FR-SA-002: Issue neighborhood alerts
- FR-SA-003: Report lost pets
- FR-SA-004: Coordinate neighborhood watch

### 5. Mutual Aid
- FR-MA-001: Post help requests
- FR-MA-002: Offer assistance
- FR-MA-003: Share skills and teach neighbors
- FR-MA-004: Create neighborhood tool library

## Technical Stack
- Frontend: React SPA
- Backend: ASP.NET Core Web API
- Database: SQL Server
- Real-time: SignalR for alerts
- Maps: Integration with mapping service for neighborhood boundaries

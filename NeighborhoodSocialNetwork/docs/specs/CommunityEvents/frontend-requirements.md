# Frontend Requirements - Community Events

## Pages
- **Events Calendar** (`/events`): Browse neighborhood events
- **Create Event** (`/events/create`): Organize new event
- **Event Details** (`/events/{id}`): View and RSVP

## Components
- EventCard: Display event with RSVP button
- EventCalendar: Calendar view of events
- RSVPList: Show who's attending
- PhotoGallery: Event photos after completion

## Features
- Interactive calendar
- One-click RSVP
- Attendance tracking
- Event reminders
- Photo sharing after events

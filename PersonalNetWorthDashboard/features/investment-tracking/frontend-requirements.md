# Frontend Requirements - Investment Tracking

## Pages
- Portfolio (`/investments`): Holdings list, allocation chart, performance
- Position Detail: Price history, gains/losses, performance

## Components
- PortfolioSummary: Total value, total return, day change
- AllocationChart: Donut chart by asset class or sector
- PerformanceMetrics: Returns (1M, 3M, YTD, 1Y, All)
- PositionCard: Ticker, shares, cost basis, current value, gain/loss

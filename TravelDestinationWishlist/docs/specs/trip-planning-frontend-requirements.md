# Frontend Requirements - Trip Planning

## Pages
- Trips Page (`/trips`): Upcoming, past, cancelled tabs
- Trip Detail Page (`/trips/{id}`): Itinerary, bookings, expenses

## Components
- TripCard: Destination(s), dates, countdown timer, booking status
- ItineraryBuilder: Day-by-day activity planner with time slots
- BookingVault: Confirmation numbers, tickets, reservations

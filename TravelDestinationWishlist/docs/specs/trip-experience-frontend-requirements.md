# Frontend Requirements - Trip Experience

## Pages
- Active Trip Page (`/trips/{id}/live`): Real-time logging, map view
- Experience Log (`/trips/{id}/experiences`): All logged activities

## Components
- LiveTripTracker: Current location, check-ins, quick log buttons
- AttractionCard: Name, photos, rating, notes, cost
- ExperienceTimeline: Chronological view of trip activities

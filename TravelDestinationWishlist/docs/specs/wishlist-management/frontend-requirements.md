# Frontend Requirements - Wishlist Management

## Pages
- Wishlist Page (`/wishlist`): Grid of destinations, priority sorting, add button
- Destination Detail Modal: Location info, research notes, edit/remove

## Components
- DestinationCard: Cover photo, location, priority stars, budget
- PriorityDragDrop: Reorder destinations by drag and drop
- ResearchPanel: Notes, attractions, costs, feasibility

# Frontend Requirements - Bedtime Management

## Components

### BedtimeReminderSettings
- Enable/disable toggle
- Target bedtime time picker
- Reminder offset slider (30, 45, 60 min)
- Custom message input
- Do-not-disturb schedule

### BedtimeStreakCard
- Current streak display
- Best streak record
- Progress to next milestone
- Motivation message

### LateBedtimeAlert
- Alert when past target bedtime
- Sleep opportunity loss display
- Encouragement to go to bed

## Pages

### Bedtime Settings (/bedtime)
- Reminder configuration
- Streak tracker
- Bedtime consistency chart

## Interactions
- Configure reminders
- Receive push notifications
- Track consistency
- View streak history

# Frontend Requirements - Dream Journaling

## Components

### DreamEntryForm
- Dream description textarea (5000 chars)
- Emotional tone selector (positive, neutral, negative, mixed)
- Vividness slider (1-10)
- Nightmare checkbox
- Tags input
- Save button

### DreamJournalList
- List of dream entries
- Search and filter by date, tags, emotional tone
- Preview of dream description
- Click to view full entry

### DreamDetail
- Full dream description
- Metadata (date, emotional tone, vividness)
- Associated sleep session link
- Edit and delete options

### RecurringThemes
- Identified recurring themes
- Theme frequency
- Associated emotions

## Pages

### Dream Journal (/dreams)
- Dream entry form
- Recent dreams list
- Search functionality

### Dream Analysis (/dreams/analysis)
- Recurring themes
- Emotional patterns
- REM correlation

## Interactions
- Log dream after waking
- Search dream journal
- Tag and categorize dreams
- View patterns
- Privacy-focused: user data only

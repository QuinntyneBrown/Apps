# Frontend Requirements - Pattern Detection

## Components

### PatternsList
- Detected patterns display
- Pattern type, frequency, confidence
- Insights and recommendations
- Severity indicators

### WeekdayWeekendComparison
- Side-by-side comparison chart
- Average sleep duration weekday vs weekend
- Social jetlag indicator
- Consistency recommendations

### InsomniAlertsPanel
- Insomnia indicators display
- Severity assessment
- Professional help recommendations
- Sleep hygiene tips

### PatternVisualization
- Timeline showing pattern occurrences
- Pattern strength over time
- Correlation with other factors

## Pages

### Patterns Dashboard (/patterns)
- All detected patterns
- Pattern visualizations
- Health alerts
- Recommendations

## Interactions
- View pattern details
- Acknowledge alerts
- Track pattern resolution
- Export pattern report

# Frontend Requirements - Nap Tracking

## Components

### NapLogForm
- Start time picker
- Duration input (or timer)
- Quality slider (1-10)
- Quick log button for standard nap (20 min)

### NapHistoryList
- List of logged naps
- Date, time, duration, quality
- Daily/weekly/monthly view

### NapImpactAnalysis
- Correlation chart: nap timing vs nighttime sleep
- Optimal nap time recommendation
- Impact on sleep quality

### ExcessiveNappingAlert
- Warning when napping excessively
- Possible causes
- Recommendations

## Pages

### Nap Dashboard (/naps)
- Quick log nap button
- Recent naps
- Nap patterns
- Impact analysis

## Interactions
- Quick log nap (one-click for standard nap)
- View nap history
- Analyze nap impact
- Get recommendations for optimal napping

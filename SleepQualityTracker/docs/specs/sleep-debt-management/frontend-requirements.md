# Frontend Requirements - Sleep Debt Management

## Components

### DebtMeter
- Visual gauge showing current debt
- Color-coded by severity (green=none, yellow=mild, orange=moderate, red=severe/critical)
- Numerical debt in hours and minutes

### DebtHistoryChart
- Area chart showing debt accumulation/repayment over time
- X-axis: dates, Y-axis: debt hours
- Color zones for severity levels
- Annotations for critical points

### RecoveryRecommendations
- Suggested extra sleep needed
- Recovery timeline estimate
- Tips for catching up on sleep

### DebtAlerts
- Warning when approaching critical levels
- Celebration when debt repaid
- Daily debt summary

## Pages

### Debt Dashboard (/debt)
- Current debt meter
- History chart
- Recovery plan
- Health warnings if critical

## Interactions
- View debt trend
- See impact of each night's sleep
- Get recovery recommendations
- Export debt report

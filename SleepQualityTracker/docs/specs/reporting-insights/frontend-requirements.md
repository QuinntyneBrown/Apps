# Frontend Requirements - Reporting & Insights

## Components

### WeeklyReportCard
- Week summary stats
- Average duration and quality
- Achievements this week
- Top 3 recommendations

### MonthlyReportDashboard
- Month overview
- Trend charts
- Patterns identified
- Progress vs previous month

### InsightsList
- Personalized insights display
- Priority indicator (high, medium, low)
- Expected benefit
- Action button for each insight

### ExportDataPanel
- Date range selector
- Format selector (CSV, PDF)
- Export button
- Download history

### RecommendationCards
- Actionable recommendations
- Category (habit, environment, schedule, etc.)
- Expected impact
- Track implementation

## Pages

### Reports Dashboard (/reports)
- Latest weekly report
- Monthly summary
- Insights feed
- Export data

### Insights Center (/insights)
- All insights sorted by priority
- Filter by category
- Implementation tracking
- Impact measurement

## Interactions

### View Weekly Report
1. Navigate to reports dashboard
2. See current week summary
3. Drill into details
4. View previous weeks

### Generate Custom Report
1. Select date range
2. Click generate
3. View comprehensive report
4. Option to export

### Act on Insights
1. View insight recommendation
2. Read details and expected benefit
3. Mark as "Implementing"
4. Track progress

### Export Data
1. Select date range
2. Choose format (CSV/PDF)
3. Click export
4. Download file

## Visualizations
- Summary cards with key metrics
- Trend line charts
- Comparison bars (this week vs last week)
- Achievement badges
- Priority indicators

## Notifications
- Weekly report ready notification (Mondays)
- New insights available
- Export completed
- Insight impact achieved

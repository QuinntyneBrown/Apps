# Requirements - Conversation Starter App

## Overview
Provide conversation prompts and questions to deepen relationships and spark meaningful discussions across various contexts (romantic, family, friends, professional).

## Core Features

### 1. Prompt Generation
- FR-PG-001: Generate prompts based on category and difficulty
- FR-PG-002: Filter prompts by context (romantic/family/friends/professional)
- FR-PG-003: Skip and get new prompt
- FR-PG-004: Favorite prompts for later use
- FR-PG-005: Create custom prompts

### 2. Conversation Tracking
- FR-CT-001: Mark prompt as used when conversation starts
- FR-CT-002: Rate conversation quality after use
- FR-CT-003: Add notes about insights from conversation
- FR-CT-004: Track conversation duration

### 3. Collections
- FR-COL-001: Create themed prompt collections
- FR-COL-002: Share collections with others
- FR-COL-003: Browse community collections
- FR-COL-004: Add prompts to collections

### 4. Progress & Insights
- FR-PI-001: Track usage streaks
- FR-PI-002: Progress through difficulty levels
- FR-PI-003: View conversation history
- FR-PI-004: Get personalized recommendations

## Technical Requirements
- Frontend: React SPA with offline support
- Backend: ASP.NET Core API
- Database: SQL Server
- Prompt library: Minimum 500 curated prompts

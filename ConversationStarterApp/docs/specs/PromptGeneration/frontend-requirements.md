# Frontend Requirements - Prompt Generation

## Pages
- **Home** (`/`): Main prompt display with generate button
- **Favorites** (`/favorites`): Saved prompts
- **Collections** (`/collections`): Browse and create collections

## Components
- PromptCard: Display prompt with actions
- CategoryFilter: Filter by category/context/difficulty
- GenerateButton: Get new prompt
- FavoriteButton: Save prompt

## Features
- Swipe gestures for skip (mobile)
- Quick favorite
- Share prompt via link
- Offline prompt caching

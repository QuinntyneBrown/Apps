# Nutrition Tracking - Frontend Requirements

## 1. Overview

Provides visual nutrition dashboards, progress tracking, and goal management with charts and trends.

## 2. Key Pages

### 2.1 Nutrition Dashboard
- Today's intake vs goals
- Macro breakdown (pie/donut charts)
- Progress bars for each nutrient
- Quick log meal button
- Weekly summary

### 2.2 Nutrition Trends
- Line charts for weekly/monthly trends
- Comparison to goals over time
- Averages and statistics
- Filter by date range

### 2.3 Goal Management
- Create/edit nutrition goals
- Goal templates (weight loss, muscle gain, etc.)
- Active goal display
- Goal history

## 3. Components

### 3.1 MacroCircle
- Circular progress indicator
- Shows calories/protein/carbs/fat
- Color-coded (green = goal met, yellow = close, red = exceeded)

### 3.2 NutritionChart
- Line chart for trends
- Bar chart for daily breakdown
- Pie chart for macro distribution

### 3.3 LogMealForm
- Search recipes
- Select meal type
- Adjust servings
- Quick add from recent

### 3.4 GoalCard
- Goal name and targets
- Start/end dates
- Progress this week
- Edit/deactivate actions

## 4. Visualizations

- Pie charts for macro breakdown
- Line charts for trends
- Progress bars for nutrients
- Heatmap calendar for compliance
- Sparklines for quick trends

## 5. Features

- Real-time goal comparison
- Streak tracking
- Weekly reports
- Export nutrition data
- Meal suggestions based on remaining macros

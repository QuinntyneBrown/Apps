# Letter Management - Frontend Requirements

## UI Components

### 1. Letter Composer
- Rich text editor
- Title input
- Mood selector
- Tags input
- Photo/attachment upload
- Save draft button
- Schedule delivery button

### 2. Schedule Delivery Modal
- Date picker (future dates only)
- Delivery time selector
- Reminder preferences
- Confirm button

### 3. Letter Reading Experience
- Full-screen reading mode
- Original writing context display
- Time elapsed calculation
- Add reflection button
- Archive button

### 4. Letter Timeline
- Chronological letter list
- Filter by status (scheduled, delivered, read)
- Search functionality
- Visual timeline view

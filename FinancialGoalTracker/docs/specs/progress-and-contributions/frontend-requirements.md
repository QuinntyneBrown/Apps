# Frontend Requirements - Progress and Contributions

## Pages/Views
### 1. Contribution History (`/goals/{id}/contributions`)
- ContributionsList (timeline view)
- AddContributionButton
- ContributionChart (over time)

### 2. Add Contribution Modal
- Amount input, date picker, source dropdown
- One-time vs Recurring toggle

## UI Components
- ProgressBar: Visual progress with percentage
- ContributionCard: Shows amount, date, source, type badge
- QuickContribution: Fast entry buttons ($10, $25, $50, $100, Custom)

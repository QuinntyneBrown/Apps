# Recommendations Engine - Frontend Requirements

## Components

### RecommendationsList
- Prioritized list of recommended screenings
- Due status badges (upcoming, due, overdue)
- Educational info expandable sections
- Quick schedule action buttons

### RecommendationCard
- Screening type and description
- Why recommended explanation
- Due date and urgency indicator
- Schedule button

### ScreeningEducation
- Description of screening
- Preparation information
- What to expect
- Linked clinical guidelines

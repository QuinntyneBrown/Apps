# Linking - Frontend Requirements

## Components
- `<KnowledgeGraph />` - Interactive graph visualization (D3.js)
- `<BacklinksPanel />` - Show pages linking here
- `<RelatedNotes />` - Suggested connections
- `<LinkInserter />` - [[WikiLink]] autocomplete

## Features
- Force-directed graph layout
- Node clustering by topic
- Link strength visualization
- Navigate by clicking nodes
- Filter by note type/tag

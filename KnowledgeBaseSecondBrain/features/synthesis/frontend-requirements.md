# Synthesis - Frontend Requirements

## Components
- `<ProgressiveSummarization />` - Layer highlighting tool
- `<InsightsDashboard />` - Synthesized knowledge view
- `<EvergreenNotes />` - Permanent notes list
- `<ThemeDetector />` - Topic clustering view

## Features
- Multi-layer highlighting (bold, highlight, super-highlight)
- Automated summary generation
- Cross-note connections
- Evergreen note development

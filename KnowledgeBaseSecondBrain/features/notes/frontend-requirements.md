# Notes - Frontend Requirements

## Components
- `<QuickCapture />` - Floating capture widget
- `<NoteEditor />` - Rich text editor
- `<NoteList />` - Searchable note list
- `<NoteView />` - Rendered note display

## Features
- Markdown editor with live preview
- Auto-save every 10 seconds
- Note templates library
- Tag autocomplete
- Related notes sidebar

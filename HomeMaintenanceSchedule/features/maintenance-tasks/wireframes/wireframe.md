## Mobile Responsive Notes

- Dashboard: Vertical stacking of cards
- Task list: Full width cards with collapsible details
- Forms: Single column layout
- Calendar: Week view default on mobile
- Kanban: Horizontal swipe between columns
- Bottom navigation bar for main sections
- Floating action button for "+ New Task"

## Accessibility Notes

- All interactive elements keyboard accessible (Tab navigation)
- ARIA labels on all buttons and form fields
- Screen reader announcements for status changes
- High contrast mode support
- Minimum touch target size: 44x44px
- Focus indicators on all interactive elements
- Alternative text for all icons and images

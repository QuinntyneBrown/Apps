## Mobile Responsive Notes

- Provider cards stack vertically on mobile
- Contact buttons (call, email) prominent and large
- Filter panel becomes bottom sheet
- Swipe gestures for card actions
- Click-to-call phone numbers native
- Simplified service form for mobile input
- Star rating uses larger touch targets

## Accessibility Notes

- ARIA labels on all star ratings
- Keyboard navigation for provider cards
- Screen reader announces rating changes
- High contrast mode for ratings
- Alt text for all icons
- Focus indicators on contact buttons

# Frontend Requirements - Gift Ideas

## Components
- Idea Bank: Grid of gift idea cards
- Add Idea Form: Quick entry for gift ideas
- Sharing Interface: Share with family members
- Claim Button: Mark idea as "I'll buy this"

## User Flows
1. Add Idea: Click "Add Idea" → Fill details → Save
2. Share Idea: Select idea → Choose recipients → Send
3. Claim Idea: View shared idea → Click "I'll buy this" → Confirm

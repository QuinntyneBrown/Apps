# Frontend Requirements - Recipient Management

## Components
- Recipient List: Cards with photos and key info
- Add/Edit Recipient Form: Personal info and preferences
- Preference Manager: Tags and interests
- Size Reference Card: Quick lookup for shopping

## User Flows
1. Add Recipient: Click "Add" → Fill form → Save
2. Update Preferences: Select recipient → Edit preferences → Save
3. Record Sizes: Select recipient → Enter sizes → Save

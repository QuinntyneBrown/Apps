# Frontend Requirements - Portfolio Management

## Components
- PortfolioValue: Large total value display with 24h change
- AllocationChart: Pie chart of holdings by coin
- PerformanceChart: Line chart of portfolio value over time
- HoldingsList: Table of all holdings with prices

## Pages
- /dashboard - Main portfolio overview

## Features
- Real-time value updates
- Interactive charts with tooltips
- Percentage and absolute value displays
- Color-coded gains/losses

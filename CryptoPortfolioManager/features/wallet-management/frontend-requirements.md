# Frontend Requirements - Wallet Management

## Components
- WalletList: Display all wallets with balances
- AddWalletForm: Add wallet by address or exchange connection
- WalletCard: Individual wallet display with sync status
- SyncButton: Manual sync trigger

## Pages
- /wallets - Wallet management dashboard

## Features
- Real-time balance updates via WebSocket
- Wallet type icons and labels
- Manual and auto-sync indicators
- Delete confirmation dialog

# Impact Tracking - Frontend Requirements

## User Interface Components

### 1. Impact Dashboard
**Components**:
- Goal progress visualization
- Milestones achieved display
- Impact metrics charts
- Achievement badges

### 2. Set Goal Form
**Form Fields**:
- Target amount (currency input)
- Tax year (dropdown)
- Goal type (dropdown)

### 3. Impact Visualization
**Charts**:
- Progress bar to goal
- Giving trends over time
- Organizations impact breakdown

# Search & Discovery - Frontend Requirements

## Components

### Global Search
**Component**: `<GlobalSearch />`

**Features**:
- Search bar in header
- Instant results dropdown
- Keyboard shortcut (Cmd/Ctrl+K)
- Recent searches
- Advanced search filters

### Search Results
**Route**: `/search`

**Features**:
- Ranked results list
- Result snippets with highlights
- Filter sidebar (namespace, category, date)
- Sort options
- Pagination

### Discovery Dashboard
**Route**: `/discover`

**Features**:
- Popular pages widget
- Recent changes feed
- Random page button
- Orphan pages alert
- Stub pages list

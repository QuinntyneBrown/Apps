# Organization - Frontend Requirements

## Components

### Namespace Tree
**Component**: `<NamespaceNavigator />`

**Features**:
- Collapsible tree structure
- Drag-and-drop to move pages
- Page count badges
- Create child namespace
- Namespace icon customization

### Category Browser
**Route**: `/categories`

**Features**:
- All categories list
- Pages per category
- Multi-select filtering
- Category hierarchy visualization
- Merge categories tool

### Page Organizer
**Component**: `<PageOrganizer />`

**Features**:
- Change namespace dropdown
- Multi-category selector
- Tag input
- Parent page selector

# Templates - Frontend Requirements

## Components

### Template Library
**Route**: `/templates`

**Features**:
- Grid of template cards
- Template preview
- Use template button
- Edit template
- Usage statistics

### Template Selector
**Component**: `<TemplateSelector />`

**Features**:
- Modal with template options
- Preview before applying
- Variable input form
- Create without template option

### Template Editor
**Route**: `/templates/{id}/edit`

**Features**:
- Rich text editor
- Variable syntax {{variable}}
- Preview with sample data
- Usage guidelines field

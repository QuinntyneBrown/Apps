# Bankroll Management - Frontend

## Pages

### Bankroll Dashboard (`/bankroll`)
- Current bankroll (large display)
- All-time high
- Profit/loss this month
- Bankroll chart over time
- Deposit/Withdraw buttons
- Transaction history

### Deposit/Withdraw Modal
- Amount input
- Description/notes
- Update balance confirmation

## UI Components
BankrollChart: Line graph showing bankroll over time with milestones marked
TransactionList: Chronological list of deposits, withdrawals, session results

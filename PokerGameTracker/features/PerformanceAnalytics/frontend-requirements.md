# Performance Analytics - Frontend

## Pages

### Analytics Dashboard (`/analytics`)
- Time period selector (30d, 90d, year, all)
- Key metrics:
  - Hourly rate
  - Win rate
  - Total sessions
  - Total profit
- Performance charts:
  - Profit trend over time
  - Win rate by game type
  - Session length vs profitability
- Best/worst sessions
- Strengths & weaknesses

## UI Components
PerformanceMetricCard: Displays metric with trend indicator
ProfitChart: Line/bar chart of cumulative profit
SessionHeatmap: Calendar view of session profitability

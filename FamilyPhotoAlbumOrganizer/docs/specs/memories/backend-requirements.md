# Memories - Backend Requirements

## Overview
Curated memory collections and anniversary detection.

## Features
- "On This Day" memories
- Anniversary detection (1, 5, 10+ years)
- Year-in-review generation
- Memory sharing

## Commands
- CreateMemoryCommand
- ShareMemoryCommand
- DetectAnniversaryCommand

## Events
- MemoryCreated
- MemoryShared
- AnniversaryDetected

## Background Jobs
- Daily memory generation
- Anniversary scanning

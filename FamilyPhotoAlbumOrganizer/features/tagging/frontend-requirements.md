# Tagging - Frontend Requirements

## Components
- Person tagging interface with face detection overlay
- Location picker with map integration
- Event selector dropdown
- Custom tag input with autocomplete
- Tag management panel

## Workflows
1. Tag Person: Click photo → Draw box → Select/add person → Save
2. Tag Location: Select photo → Choose location from map → Save
3. Tag Event: Select photos → Choose event → Apply to all
4. Add Custom Tag: Type tag → Press enter → Tag added

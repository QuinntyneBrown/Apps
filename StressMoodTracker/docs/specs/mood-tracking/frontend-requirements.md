# Frontend Requirements - Mood Tracking

## Components
- Mood Check-in: Quick slider + emotion tags
- Mood History: Timeline/calendar view
- Mood Chart: Line graph of mood over time
- Streak Display: Achievement badge
- Swing Alerts: Notification of significant changes

## User Flows
1. Log Mood: Open app → Slide mood rating → Select emotions → Add note → Save
2. View History: Navigate to history → See mood timeline → Tap for details
3. Celebrate Streak: Achieve milestone → See celebration screen

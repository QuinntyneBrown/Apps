# Digital Will - Frontend Requirements

## Pages & Components

### Digital Will Builder
**Route**: `/will`

**Components**:
- Will compilation dashboard
- Version history
- Export to PDF button
- Legal status indicator
- Share with attorney feature

### Will Compilation Wizard
**Steps**:
1. Account Coverage Review
2. Beneficiary Assignments
3. Instruction Compilation
4. Legal Information
5. Review & Finalize

### Ethical Will Editor
**Component**: `<EthicalWillEditor />`

**Features**:
- Rich text editor
- Personal wishes sections
- Values and beliefs documentation
- Life lessons recording
- Family message sections

### Final Messages
**Component**: `<FinalMessageComposer />`

**Features**:
- Encrypted message composer
- Multiple recipients
- Delivery condition selector
- Preview before saving
- Seal and encrypt button

## UI/UX
- Formal, respectful tone
- Version comparison view
- Legal disclaimer display
- Attorney integration options
- Download/print formatting

- **Summary Header**: Count of medications and interactions

- **Severity Sections**: Grouped by critical, moderate, minor
- **Interaction Cards**: Detailed information for each interaction

- Critical: Red background
- Moderate: Yellow/Orange background
- Minor: Blue background
- None: Green success message

## Responsive Considerations

### Mobile View (<768px)
- Stack cards vertically
- Full-width components
- Simplified navigation
- Swipe gestures for actions
- Bottom sheet dialogs
- Collapsible sections

### Tablet View (768-1024px)
- Two-column card layout where appropriate
- Preserved spacing
- Touch-friendly buttons
- Side panels for filters

### Desktop View (>1024px)
- Multi-column layouts
- Side-by-side details
- Hover states
- Keyboard shortcuts
- Expanded information density

# Recipe Management - Frontend

## Pages

### Recipe Library (`/recipes`)
- Grid of recipe cards
- Filter by style
- Sort by rating, brew count
- Clone/Edit/Delete actions

### Recipe Builder (`/recipes/new`)
- Beer name and style selector
- Target ABV/IBU inputs
- Grain bill builder (add/remove grains with quantities)
- Hop schedule (addition times and amounts)
- Yeast selector
- Batch size calculator
- Expected stats preview (OG, FG, ABV, IBU)

## UI Components
RecipeCard: Shows beer name, style, ABV, IBU, last brewed date, success rating

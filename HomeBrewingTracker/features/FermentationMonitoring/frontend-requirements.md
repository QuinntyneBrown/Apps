# Fermentation Monitoring - Frontend

## Pages

### Fermentation Dashboard (`/fermentations/{id}`)
- Days since pitch
- Temperature chart over time
- Expected completion date
- Activity observations log
- Gravity check reminders

### Temperature Log
- Manual temperature entry
- Target range indicator
- Alerts if out of range
- Historical chart

## UI Components
FermentationTimeline: Visual timeline showing pitch, active fermentation, completion
TemperatureChart: Line graph of temperature over fermentation period

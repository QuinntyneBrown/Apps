# Brew Day Tracking - Frontend

## Pages

### Active Brew Session (`/brew-sessions/{id}/active`)
- Timer showing elapsed time
- Current step indicator (Mash, Boil, Cool, Ferment)
- Temperature input for mash
- Gravity reading input
- Notes field
- Checklist of brew day tasks

### Brew Session Summary
- Recipe brewed
- Start/end times
- OG reading and efficiency achieved
- Issues encountered
- Photos of brew day

## UI Components
BrewTimer: Elapsed time with step milestones
GravityInput: Hydrometer reading entry with temperature correction

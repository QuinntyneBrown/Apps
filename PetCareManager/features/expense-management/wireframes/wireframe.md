## Design Notes

### Color Coding
- **Success/Approved**: Green (#10B981)
- **Pending**: Yellow/Amber (#F59E0B)
- **Denied**: Red (#EF4444)
- **Draft**: Gray (#6B7280)
- **Primary Actions**: Blue (#3B82F6)

### Icons
- Veterinary Care: Medical cross (+)
- Food: Bowl
- Grooming: Scissors
- Boarding: House
- Supplies: Shopping bag
- Insurance: Shield

### Interactive Elements
- All buttons have hover states
- Cards have subtle shadows
- Forms have real-time validation
- Tables support row selection
- Modals have backdrop overlay
- Charts are interactive (hover for details)

### Accessibility
- Proper heading hierarchy
- ARIA labels on icon buttons
- Keyboard navigation support
- Focus indicators on all interactive elements
- Color not sole indicator of status (use icons/text)
- Sufficient color contrast ratios

### Responsive Breakpoints
- Mobile: < 768px (stacked layout)
- Tablet: 768px - 1024px (2-column grids)
- Desktop: > 1024px (multi-column layouts)

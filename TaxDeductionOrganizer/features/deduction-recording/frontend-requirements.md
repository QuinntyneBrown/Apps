# Deduction Recording - Frontend Requirements
## Key Components
1. Quick Entry Form: Amount, category, vendor, date, receipt upload
2. Deduction List: Filterable, sortable, with flag indicators
3. Category Selector: IRS-compliant dropdown with descriptions
4. Review Queue: Flagged deductions needing approval
## API Integration
- POST /api/deductions, PUT /api/deductions/{id}/flag

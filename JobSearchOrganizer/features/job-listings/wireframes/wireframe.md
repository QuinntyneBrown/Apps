## Notes

### Design Principles
- **Clarity**: Clear status indicators and priority levels
- **Efficiency**: Quick actions available on cards
- **Flexibility**: Grid and list views for different preferences
- **Responsiveness**: Mobile-friendly layouts
- **Accessibility**: High contrast, keyboard navigation, ARIA labels

### Color Coding
- **Priority**: Red (High), Yellow (Medium), Green (Low)
- **Status**: Blue (Discovered), Green (Saved), Gray (Archived)
- **Deadlines**: Red (<3 days), Orange (<7 days), Yellow (<14 days), Green (>14 days)

### Interactions
- Click card to view details
- Hover to reveal quick actions
- Drag to reorder or change status
- Bulk select with checkboxes
- Filter chips can be clicked to remove

### Progressive Disclosure
- Show essential info on cards
- Full details on dedicated page
- Expandable sections for description
- Collapsible filters on mobile

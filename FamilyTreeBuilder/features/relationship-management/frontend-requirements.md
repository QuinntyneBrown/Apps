# Relationship Management - Frontend Requirements

## UI Components

### 1. Add Relationship Form
- Select person 1 (dropdown)
- Select person 2 (dropdown)
- Relationship type (dropdown)
- Start date (optional)
- Save button

### 2. Family Tree Visualization
- Interactive SVG tree diagram
- Zoom and pan controls
- Click nodes to view details
- Different layouts (pedigree, fan, descendant)

### 3. Relationship List
- Table of all relationships
- Filter by type
- Edit/delete actions

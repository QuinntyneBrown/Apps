# Person Management - Frontend Requirements

## User Interface Components

### 1. Add Person Form
**Form Fields**:
- Full Name* (text)
- Given Name* (text)
- Surname* (text)
- Gender* (dropdown)
- Birth Date (date picker)
- Birth Place (text with location autocomplete)
- Living Status (toggle)
- Death Date (if deceased, date picker)
- Death Place (text)
- Occupation (text)
- Biography (rich text editor)

### 2. Person Profile Page
**Sections**:
- Header with name, dates, photo
- Vital statistics (birth, death, age)
- Life events timeline
- Family relationships
- Photos and media
- Stories and memories
- Documents and sources
- Edit button

### 3. Person Card (List View)
**Display**:
- Profile photo thumbnail
- Name and dates
- Life span indicator
- Relationship to user
- Quick actions (View, Edit)

## Responsive Design
- Mobile: Single column, expandable sections
- Tablet: Two columns
- Desktop: Multi-column with sidebar

# Tracking - Frontend Requirements

## Components
- `<ActiveRoutine />` - Current routine in progress
- `<ActivityTimer />` - Countdown timer for activity
- `<CompletionTracker />` - Check off completed activities
- `<StreakDisplay />` - Show current streak
- `<ProgressStats />` - Analytics dashboard

## Features
- Step-by-step activity guide
- Timer with notifications
- Quick skip/modify options
- Completion celebration animations
- Streak badges and achievements

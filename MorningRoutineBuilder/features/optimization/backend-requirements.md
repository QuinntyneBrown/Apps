# Optimization - Backend Requirements

## API Endpoints
- GET /api/optimization/suggestions - Get improvement suggestions
- POST /api/optimization/ab-test - Start A/B test
- GET /api/optimization/insights - Get routine insights

## Domain Events
- OptimizationSuggested
- RoutineVariationTested
- ImprovementImplemented

## Business Logic
- Identify bottleneck activities
- Suggest duration adjustments
- Detect energy patterns
- Recommend activity reordering
- Calculate optimal routine length

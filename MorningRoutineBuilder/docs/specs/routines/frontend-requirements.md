# Routines - Frontend Requirements

## Components
- `<RoutineBuilder />` - Drag-and-drop routine creator
- `<ActivitySelector />` - Choose from activity library
- `<RoutineTimeline />` - Visual timeline view
- `<RoutineEditor />` - Edit existing routine

## Features
- Drag-and-drop activity ordering
- Duration sliders
- Visual timeline preview
- Template selection
- Routine variations (weekday/weekend)

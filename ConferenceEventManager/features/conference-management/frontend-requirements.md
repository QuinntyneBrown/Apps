# Frontend Requirements - Conference Management

## Key Pages
1. **Conferences List** - Grid/list view of all conferences
2. **Conference Detail** - Full conference info, schedule, sessions, contacts
3. **Add Conference** - Registration form with details
4. **Schedule Planner** - Interactive session scheduler

## Features
- Register for new conferences
- Upload confirmation emails
- Plan session schedules with conflict detection
- Mark conferences as attended
- Rate overall experience
- Export to calendar
- Track expenses related to conference

## Components
- ConferenceCard, ConferenceForm, ScheduleBuilder, SessionTimeline

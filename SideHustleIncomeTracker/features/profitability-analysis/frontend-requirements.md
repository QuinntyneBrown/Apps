# Profitability Analysis - Frontend Requirements

## Key Components
1. **P&L Dashboard**: Revenue, expenses, net profit by stream
2. **Profitability Charts**: Trend lines, profit margin visualization
3. **Stream Comparison**: Side-by-side performance metrics
4. **Milestone Tracker**: Achievement badges and history
5. **Period Selector**: Monthly, quarterly, yearly, custom range

## API Integration
- POST /api/profitability/calculate, GET /api/profitability/comparison

# Frontend Requirements - Resume Management

## Pages
- **Resume Versions** (`/resumes`): Manage all resume versions
- **Resume Editor** (`/resumes/{id}/edit`): Edit resume content
- **Tailor Resume** (`/resumes/{id}/tailor`): Customize for job
- **ATS Optimizer** (`/resumes/{id}/optimize`): Check ATS compatibility

## Components
- ResumeEditor: Rich text editor with formatting
- VersionSelector: Switch between resume versions
- KeywordHighlighter: Show matched keywords
- ATSScoreCard: Display optimization score
- ExportOptions: PDF, Word, HTML format selection

## Features
- Drag-and-drop section reordering
- Achievement selector from library
- Real-time keyword matching
- ATS score visualization
- One-click export to multiple formats

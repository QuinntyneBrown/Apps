## Interaction Notes

1. **Checkbox Click**: Quick purchase (uses estimated price)
2. **Checkbox Long Press**: Show purchase dialog with price input
3. **Item Swipe Right (Mobile)**: Mark as purchased
4. **Item Swipe Left (Mobile)**: Delete item
5. **Quantity +/- Buttons**: Immediate update with haptic feedback
6. **Item Name Click**: Edit inline (desktop) or modal (mobile)
7. **Category Header Click**: Collapse/expand category
8. **Drag Item**: Reorder within category
9. **Pull to Refresh**: Sync latest changes
10. **Offline Mode**: Queue changes, show sync icon

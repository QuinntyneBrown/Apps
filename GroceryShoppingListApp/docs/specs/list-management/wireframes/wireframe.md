## Interaction Notes

1. **List Card Hover State**: Show quick action buttons (Share, Complete, Archive, Delete)
2. **List Card Click**: Navigate to list details page
3. **Drag and Drop**: Support reordering lists (saves as user preference)
4. **Swipe Actions (Mobile)**:
   - Swipe right: Mark complete
   - Swipe left: Delete/Archive options
5. **Pull to Refresh (Mobile)**: Refresh list data
6. **Long Press (Mobile)**: Show context menu with actions
7. **Keyboard Shortcuts (Desktop)**:
   - `N`: New list
   - `S`: Search
   - `F`: Toggle filters
   - `1/2/3`: Switch between Active/Completed/Archived tabs

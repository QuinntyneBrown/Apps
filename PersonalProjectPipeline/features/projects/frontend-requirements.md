# Projects - Frontend Requirements

## Components
- `<ProjectBoard />` - Kanban board view
- `<ProjectList />` - List/table view
- `<ProjectDetails />` - Project detail page
- `<ProjectForm />` - Create/edit form

## Features
- Drag-and-drop status changes
- Filter by status, priority, tags
- Sort by various fields
- Quick actions (start, pause, complete)
- Progress visualization

# Timeline - Frontend Requirements

## Components
- `<GanttChart />` - Gantt timeline view
- `<MilestoneMarkers />` - Key milestone indicators
- `<CalendarView />` - Calendar integration
- `<CriticalPath />` - Highlight critical path

## Features
- Interactive Gantt chart
- Zoom levels (day/week/month)
- Drag to reschedule
- Milestone markers
- Today indicator line

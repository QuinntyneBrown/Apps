# Tasks - Frontend Requirements

## Components
- `<TaskList />` - Filterable task list
- `<TaskForm />` - Create/edit tasks
- `<TaskDependencies />` - Dependency visualization
- `<TaskTimeline />` - Gantt chart view

## Features
- Nested subtasks
- Drag-and-drop reordering
- Quick add task
- Bulk actions
- Dependency arrows in timeline
